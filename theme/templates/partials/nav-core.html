Validated via safe_href (https / mailto / tel / root-relative / anchor only). No active state — there is no slug to compare against PAGE - { label, children: [...] } → dropdown group whose children use the same three shapes above Preference order: anchor > slug > href. Use slug for routed pages whenever possible (active state, language flexibility); use href only when slug-routing cannot express the target. Editor can reorder, relabel, show/hide items without touching this template. New routed pages still need a routes.php entry from the developer first; the editor only decides whether (and where) the page appears in the nav. #}
